## VSAって何？

VRChatで撮った写真、こんな悩みありませんか？

- 写真がフォルダに散乱して整理できない
- どこで撮ったか、誰と撮ったか思い出せない
- 大量のスクショでストレージがパンパン

**VSA**はこれらの悩みをすべて解決します！

---

## インストール方法

### 必要なもの

- **Windows 10/11** (64bit)
- **500MB**以上の空き容量

### 手順

1. [**ダウンロードページ**](https://github.com/JunseiOgawa/VSA/releases)へ移動
2. 最新版の`VSA-Setup-vX.X.X.exe`をダウンロード
3. ダウンロードしたファイルを実行
4. 画面の指示に従ってインストール
5. デスクトップのアイコンからVSAを起動！

## できること

### VSA - 写真管理アプリ

| 機能 | 説明 |
|------|------|
| **ギャラリー表示** | 撮った写真を一覧で見やすく表示 |
| **自動取り込み** | VRChatの写真を自動で取り込み |
| **撮影情報を自動保存** | ワールド名、撮影日時などを自動記録 |
| **写真を圧縮** | 高画質のまま容量を30-50%削減 |
| **お気に入り機能** | 大切な写真にマークを付けて管理 |
| **テーマ切替** | ダーク/ライトモード選択可能 |

### VDI - 写真ビューワ

| 機能 | 説明 |
|------|------|
| **拡大・縮小** | 細部まで確認できる高精度ズーム |
| **構図ガイド** | 三分割法などの構図ガイド表示 |
| **ピント確認** | ピントが合っている場所をハイライト |
| **ヒストグラム** | 写真の明るさ分布を確認 |

## 使い方

### はじめての設定

1. **写真フォルダを設定する**
   - サイドバーの「ゲーム側設定」をクリック
   - 「フォルダを選択」でVRChatの写真保存先を選ぶ
   - 通常は `C:\Users\あなたの名前\Pictures\VRChat`

2. **監視をスタート**
   - 「監視開始」ボタンをクリック
   - これで新しい写真が自動で取り込まれます！

3. **ギャラリーで確認**
   - サイドバーの「ギャラリー」をクリック
   - 取り込まれた写真が一覧表示されます

### よく使う操作

| やりたいこと | 方法 |
|--------------|------|
| 写真を大きく見る | サムネイルをクリック |
| お気に入り登録 | お気に入りボタンをクリック |
| 写真を圧縮 | 設定からJXL圧縮を有効化 |

## よくある質問

### 写真が表示されない

1. フォルダが正しく設定されているか確認
2. 「監視開始」ボタンを押したか確認
3. アプリを再起動してみる

### 容量を節約したい

設定画面でJXL圧縮を有効にすると、写真の容量を30-50%削減できます。
画質はほとんど変わりません！

### VRChatプレイ中に動作が重くなる？

大丈夫！VRChatが起動中は自動的に処理を一時停止します。
