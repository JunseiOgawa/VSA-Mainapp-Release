# VSAの基本的な使い方

VSA（VRChat Screenshot Archive）は、VRChatで撮影した写真を効率的に管理・圧縮・クラウド保存するための総合的なアプリケーションです。

## 11の主要機能

VSAには、以下の11の機能が備わっています：

### 1. ギャラリービュー（Gallery View）

インポートした写真をグリッド表示またはテーブル表示で閲覧・管理できます。

- グリッド表示：写真をサムネイルで一覧表示
- テーブル表示：ファイル情報をリスト形式で表示
- ズーム：Ctrl + =/- でサムネイルサイズを調整
- ソート：作成日、ワールド名、ファイル名などで並べ替え

詳細：[ギャラリー操作ガイド](docs/ja/gallery-guide.md)

### 2. フォルダ監視・インポート（Import Folder）

VRChatの写真フォルダを自動監視し、新しい写真を自動的にアプリケーションに取り込みます。

- フォルダを追加：複数のフォルダを同時に監視
- 自動インポート：新しい写真が撮影されると自動検知
- 進捗表示：インポート状況をリアルタイムで確認

詳細：サイドバーの「インポート」セクション

### 3. お気に入い管理（Favorites）

大切な写真にマークを付けて、専用ビューで一元管理できます。

- お気に入い追加：写真の星アイコンをクリック
- 専用ビュー：お気に入い写真だけを表示
- Google Drive連携：自動的にクラウドにバックアップ

詳細：[お気に入い機能ガイド](docs/ja/favorites-guide.md)

### 4. 設定画面（Settings）

アプリケーションの各種設定を8つのパネルで管理できます。

- 外観：テーマ、言語、カラースキーム
- アニメーション：UI効果のオン/オフ
- ファイル：自動保存、履歴管理
- 通知：デスクトップ通知設定
- ショートカット：キーボードショートカットのカスタマイズ
- VDI：VDI連携機能
- お気に入い：フォルダ管理
- アップデート：自動更新設定

詳細：[設定画面完全ガイド](docs/ja/settings-guide.md)

### 5. ゲーム側設定（Game Config）

VRChatとの連携を設定し、メタデータの自動抽出を有効にします。

- VRChat監視：プロセスの自動検知
- メタデータ出力：ワールド名、ユーザー情報の自動抽出
- OSC設定：カメラパラメータの取得
- 自動機能：自動インポート、自動圧縮の設定

詳細：[ゲーム側設定ガイド](docs/ja/game-config.md)

### 6. JXL圧縮（JPEG XL Compression）

写真をJPEG XL形式で圧縮し、ストレージ容量を大幅に削減できます。

- 自動圧縮：スケジュールに基づく定期実行
- 手動圧縮：任意のタイミングで実行
- ロスレス圧縮：画質を劣化させない
- メタデータ保持：圧縮後もすべての情報が保持される

詳細：[JXL圧縮ガイド](docs/ja/jxl-compression.md)

### 7. ユーザー検索（Person Search）

写真に写っているユーザー名で検索し、特定の人物が写っている写真を素早く見つけられます。

- タグベース検索：ユーザー名をタグのように追加
- ローマ字検索：日本語をローマ字で検索可能
- AND検索：複数ユーザーで全員が写っている写真を検索
- お気に入いユーザー：よく検索するユーザーを登録

詳細：[ユーザー検索機能ガイド](docs/ja/person-search-guide.md)

### 8. ドキュメント閲覧（Documents）

アプリ内でユーザーガイドやチュートリアルを閲覧できます。

- ガイド：各機能の詳細説明
- チュートリアル：初心者向けの使い方
- FAQ：よくある質問と解答

### 9. アカウント管理（Account）

Google Driveを連携してクラウドバックアップを設定します。

- Google Drive OAuth2認証
- 自動アップロード：お気に入い追加時に自動でクラウド保存
- 手動アップロード：任意のタイミングでアップロード

詳細：[アカウント管理ガイド](docs/ja/account-guide.md)

### 10. チュートリアル（Tutorial）

初心者向けのステップバイステップガイドです。

- セットアップガイド：初回設定
- インポートガイド：フォルダ追加方法
- 基本操作：各機能の基本的な使い方

### 11. セットアップウィザード（Setup Wizard）

アプリケーション初回起動時の初期設定を行います。

- 言語とテーマ選択
- フォルダ設定
- 自動起動設定
- Google Drive連携（オプション）

詳細：[初回セットアップガイド](docs/ja/setup-wizard.md)

## 基本的なワークフロー

### 初心者向け（初回セットアップから基本操作まで）

1. **初回セットアップ**
   - アプリを起動するとセットアップウィザードが表示される
   - 言語、テーマ、VRChatのフォルダを選択
   - セットアップを完了する

2. **ゲーム側設定**
   - サイドバーの「ゲーム側設定」を開く
   - VRChat監視を有効化
   - メタデータ出力フォルダを設定

3. **写真の確認**
   - ギャラリーで撮影した写真を閲覧
   - 写真をクリックして詳細情報（ワールド名、ユーザー情報）を確認

4. **お気に入い登録**
   - 大切な写真の星アイコンをクリック
   - お気に入いビューで管理

### 上級者向け（効率的な管理と最適化）

1. **自動監視の設定**
   - ゲーム側設定で自動インポートを有効化
   - VRChat起動中に撮影すると自動的にアプリに追加

2. **JXL圧縮の自動化**
   - ゲーム側設定で自動圧縮を有効化
   - スケジュール設定：月1回、月初に自動圧縮実行
   - VRChat状態を考慮：ゲーム中の圧縮を一時停止

3. **Google Drive連携**
   - アカウント画面で「Google Driveでログイン」
   - 自動アップロードを有効化
   - お気に入い追加時に自動的にクラウド保存

4. **ユーザー検索の活用**
   - よく撮影するフレンドをお気に入いユーザーに登録
   - タグベース検索で複数ユーザーの写真を検索
   - AND検索で特定イベントの写真を効率的に見つける

5. **ショートカットのカスタマイズ**
   - 設定画面 > ショートカットで頻繁に使う操作をカスタマイズ
   - F10：スクリーンサイズ切り替え
   - F11：フルスクリーン表示
   - Ctrl+=/−：ギャラリーズーム

## ワークフロー例

### 例1: VRChat定期プレイ時（毎週末）

1. 金曜日19:00：自動監視有効、ゲーム側設定でメタデータ出力を確認
2. VRChat起動：新しい写真は自動的にアプリに追加される
3. 好きな写真にお気に入いマーク：自動的にGoogle Driveに保存
4. 月1回（月初）：JXL圧縮が自動実行、ストレージを最適化

### 例2: 大規模イベント参加（複数フレンドとの撮影）

1. イベント前：ゲーム側設定でメタデータ設定を確認
2. イベント中：複数フレンドとの写真を撮影
3. イベント後：ユーザー検索で参加者ごとの写真を確認
4. AND検索：全員が写っている写真のみをお気に入い登録
5. クラウド保存：Google Driveに全員の写真をバックアップ

### 例3: 年間アーカイブ管理

1. 毎月初：前月の写真をお気に入い登録
2. 毎月末：JXL圧縮実行で容量最適化
3. 四半期ごと：Google Driveのバックアップ確認
4. 年末：ユーザー検索で年間フレンドランキング作成
5. 翌年初：古い写真をアーカイブフォルダに移動

## ヒント

- **初めて使用する場合**：チュートリアルを実行することをお勧めします
- **ヘルプアクセス**：ヘルプボタン（?アイコン）からいつでもドキュメントを開始できます
- **設定のカスタマイズ**：設定画面でアプリケーションの動作を細かく調整できます
- **パフォーマンス最適化**：大量の写真がある場合、サムネイルサイズを小さくするとスクロール速度が向上します
- **バックアップの実施**：Google Drive連携でクラウドに定期的にバックアップすることをお勧めします
