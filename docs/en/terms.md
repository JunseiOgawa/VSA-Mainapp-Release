## 2. Service Overview

The App is a desktop application designed for managing, adding metadata to, and organizing photos taken in VRChat. Key features include:

- Automatic detection and import of VRChat screenshots
- Automatic metadata attachment to photos (world information, camera settings, etc.)
- Gallery display and management of photos
- Photo compression to JXL format
- Favorites functionality
