# フォルダのインポート

このガイドでは、VRChatの写真フォルダをVSAに追加して管理する方法を説明します。

## フォルダの追加

### 手順

1. サイドバーの「インポート」をクリック
2. 「フォルダを追加」ボタンをクリック
3. ファイルダイアログでフォルダを選択
4. 「OK」をクリック

### VRChatのデフォルトフォルダ

VRChatの写真は通常、以下の場所に保存されています:

```
%USERPROFILE%\Pictures\VRChat
```

例: `C:\Users\ユーザー名\Pictures\VRChat`

## フォルダの管理

### フォルダ一覧

追加したフォルダは一覧で表示されます。各フォルダには以下の情報が表示されます:

- フォルダパス
- 写真の数
- 最終更新日時

### フォルダの選択

複数のフォルダを選択して一括操作ができます:

1. フォルダ左側のチェックボックスをクリック
2. 複数選択も可能
3. 全選択/全解除ボタンで一括操作

### フォルダの削除

不要なフォルダを削除するには:

1. 削除したいフォルダを選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「OK」をクリック

**注意**: フォルダの削除は、VSAからの登録解除のみです。実際のファイルは削除されません。

## 自動スキャン

### 自動スキャンとは

自動スキャン機能を有効にすると、登録したフォルダに新しい写真が追加された際に、自動的にVSAに取り込みます。

### 設定方法

1. インポート画面の「自動スキャン」トグルをオンにする
2. スキャン間隔を設定（設定画面で変更可能）

### スキャン間隔

デフォルトでは1分ごとにスキャンが実行されます。設定画面でカスタマイズできます。

## 処理の実行

フォルダを追加した後、写真の処理を実行します:

1. 処理したいフォルダを選択（またはすべてのフォルダ）
2. 「処理を実行」ボタンをクリック
3. 処理完了を待つ

処理中は進捗バーで進行状況を確認できます。

## ヒント

### 複数フォルダの管理

- 年別や月別にフォルダを分けている場合、すべて追加できます
- フォルダ名で内容を識別しやすくしましょう

### パフォーマンス

- 大量の写真があるフォルダの初回スキャンには時間がかかることがあります
- 自動スキャンは増分のみをチェックするため、2回目以降は高速です

### トラブルシューティング

**フォルダが追加できない場合:**
- フォルダへのアクセス権限を確認
- パスに特殊文字が含まれていないか確認

**写真が検出されない場合:**
- 対応形式: PNG, JPG, JPEG
- サブフォルダ内の写真も自動的に検出されます
