# はじめに

VSAへようこそ！このガイドでは、VSAを初めて使用する方向けに、基本的なセットアップと使い方を説明します。

## VSAとは

VSA（VRChat Screenshot Archive）は、VRChatで撮影した写真を効率的に管理・圧縮・クラウド保存するための総合的なアプリケーションです。

### 主な特徴

- **写真の一元管理**: 複数のフォルダから写真を取り込んで一覧表示
- **高効率な圧縮**: JPEG XL形式による高品質・低容量の圧縮
- **自動処理**: 新しい写真の自動検出と圧縮
- **ユーザー検索**: フレンド名で写真を素早く検索
- **Google Drive連携**: お気に入い写真を自動クラウド保存
- **VRChat連携**: ワールド名やユーザー情報の自動抽出
- **使いやすいUI**: モダンで直感的なインターフェース

## 最初のステップ: セットアップウィザード

VSAを初めて起動すると、**セットアップウィザード**が自動的に表示されます。このウィザードに従って、基本的な設定を行います。

### セットアップウィザードの流れ

1. **ようこそ画面**: VSAの概要を確認
2. **言語とテーマ選択**: 使用言語とテーマを選択
3. **フォルダ設定**: VRChatの写真フォルダを指定
4. **自動起動設定**: Windowsスタートアップで自動起動するかを選択
5. **セットアップ完了**: メイン画面に遷移

**注意**: セットアップをスキップすることもできますが、後で設定画面から手動で設定できます。

詳細は[初回セットアップガイド](../ja/setup-wizard.md)を参照してください。

## ステップ1: ゲーム側設定

セットアップウィザード完了後、**ゲーム側設定**でVRChat連携を有効にします。これにより、写真のメタデータ（ワールド名、ユーザー情報など）が自動的に抽出されます。

### 設定手順

1. **サイドバーから「ゲーム側設定」を開く**
   - サイドバーの歯車アイコン（⚙️）をクリック
   - 左側のメニューから「ゲーム側設定」を選択

2. **VRChat監視を有効化**
   - トグルスイッチで監視をONにする
   - VSAがVRChatの起動/終了を自動検知

3. **メタデータ出力フォルダを設定**
   - VRChatのログ出力フォルダを指定
   - ワールド名やユーザー情報がここから抽出される

詳細は[ゲーム側設定ガイド](../ja/game-config.md)を参照してください。

## ステップ2: 写真フォルダの追加

VRChatの写真が保存されているフォルダをVSAに追加します。

### 手順

1. **「インポート」をクリック**
   - サイドバーの「インポート」メニューを開く

2. **「フォルダを追加」ボタンをクリック**
   - フォルダ選択ダイアログが表示される

3. **VRChatの写真フォルダを選択**
   - デフォルト: `C:\Users\[ユーザー名]\Pictures\VRChat`
   - カスタムフォルダを使用している場合はそのパスを選択

4. **インポート開始**
   - フォルダが追加され、自動的に写真のスキャンが開始されます
   - 進捗バーでインポート状況を確認できます

## ステップ3: 写真の確認

フォルダをインポートすると、ギャラリービューで写真を確認できます。

### ギャラリーの使い方

1. **「ギャラリー」をクリック**
   - サイドバーの「ギャラリー」メニューを開く

2. **サムネイルで写真を一覧表示**
   - グリッド形式でサムネイルが表示される
   - Ctrl + = でズームイン、Ctrl + − でズームアウト

3. **写真をクリックして詳細を確認**
   - 右側のサイドバーに詳細情報が表示される
   - ワールド名、撮影日時、ユーザー一覧などが表示される

詳細は[ギャラリー操作ガイド](../ja/gallery-guide.md)を参照してください。

## ステップ4: お気に入いとクラウド連携（オプション）

大切な写真をお気に入いに登録し、Google Driveに自動保存できます。

### お気に入いの追加

1. **写真の星アイコンをクリック**
   - サムネイル右上の星マークをクリック
   - または詳細サイドバーの星ボタンをクリック

### Google Drive連携（オプション）

1. **アカウント画面を開く**
   - サイドバーの「アカウント」をクリック

2. **「Google Driveでログイン」をクリック**
   - Webブラウザが自動的に開く
   - Googleアカウントで認証

3. **自動アップロード設定**
   - アカウント画面で「自動アップロード」を有効化
   - お気に入い追加時に自動的にクラウド保存されます

詳細は[お気に入い機能ガイド](../ja/favorites-guide.md)と[アカウント管理ガイド](../ja/account-guide.md)を参照してください。

## 11の主要機能の概要

VSAの全機能は[基本的な使い方](../basic-usage.md)に詳しく説明されています：

1. **ギャラリービュー**: グリッド/テーブル表示、ソート、ズーム
2. **フォルダ監視**: 自動インポート、進捗表示
3. **お気に入い**: 写真マーク、専用ビュー、Google Drive連携
4. **設定画面**: テーマ、言語、通知設定など8つのパネル
5. **ゲーム側設定**: VRChat監視、メタデータ抽出、OSC設定
6. **JXL圧縮**: 自動/手動圧縮、ロスレス圧縮
7. **ユーザー検索**: タグベース検索、ローマ字対応、AND検索
8. **ドキュメント**: アプリ内ガイド参照
9. **アカウント**: Google Drive認証、自動アップロード
10. **チュートリアル**: ステップバイステップガイド
11. **セットアップウィザード**: 初回設定アシスタント

詳細は[基本的な使い方](../basic-usage.md)を参照してください。

## トラブルシューティング

### 写真が表示されない場合

1. フォルダが正しく追加されているか確認
2. フォルダ内に写真ファイル（PNG形式）があるか確認
3. アプリケーションを再起動
4. 詳細は[FAQ](../ja/faq.md)を参照

### メタデータが取得できない場合

1. ゲーム側設定でメタデータ出力フォルダが正しく設定されているか確認
2. VRChatを起動してスクリーンショットを撮影
3. VRChatでOSC設定が有効になっているか確認
4. 詳細は[ゲーム側設定ガイド](../ja/game-config.md)を参照

### 圧縮が動作しない場合

1. VRChatが起動中でないか確認（起動中は圧縮が一時停止）
2. 出力先フォルダに十分な空き容量があるか確認
3. ゲーム側設定で自動圧縮が有効になっているか確認
4. 詳細は[JXL圧縮ガイド](../ja/jxl-compression.md)を参照
